# StoreCompleteCartRes

 **StoreCompleteCartRes**: { `data`: [`Cart`](../classes/internal.Cart.mdx) ; `type`: ``"cart"``  } \| { `data`: [`Order`](../classes/internal.Order.mdx) ; `type`: ``"order"``  } \| { `data`: [`Swap`](../classes/internal.Swap.mdx) ; `type`: ``"swap"``  }

If the cart is completed successfully, this will have the created order or the swap's details, based on the cart's type. Otherwise, it'll be the cart's details.
